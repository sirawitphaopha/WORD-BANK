## 🥇 ลำดับ 1 — ความแม่นการจัดหมวด (เรื่องใหญ่สุด · พี่กันพึ่ง AI จัดหมวด)

> พี่กันเป็นเภสัช ไม่เชี่ยวภาษา → จัดหมวดแม่น = หัวใจของคลัง · v2 พบ AI จัดตรงกันแค่ 42%

- 🔴 **N. ทบทวนหมวดทั้งคลัง → เขียนความหมายหมวดย่อย → ส่งเข้า prompt** *(ไอเดียใหม่ 21 ก.ค.)*
  ต้นเหตุที่ AI จัดไม่ตรง อาจเพราะหมวดไม่ละเอียด/สื่อความหมายไม่พอ → ให้ AI เห็นคำนิยามหมวด จัดแม่นขึ้น
- 🔴 **J. แนบตัวอย่าง 2-3 คำต่อกิ่ง ไปกับ prompt** — *แคลร์ประเมินว่าช่วยความแม่นได้มากที่สุด* · ทำคู่กับ N ยิ่งดี
- 🔴 **E. Benchmark + เฉลย** — พี่กันทำชุดคำ+เฉลยหมวด → วัด % ความแม่นรายโมเดล (เป็นตัวเลข ไม่ใช่รู้สึกเอา) · = แผนเวอร์ชัน ๓
- 🔴 **Gemini#2. ตีกรอบ "คำงาม" ให้ชัดใน prompt** (vivid verbs / คำไวพจน์ / metaphor) — ห้ามลดปริมาณสกัด
- 🔴 **K. ทักเตือนกิ่งซ้ำความหมาย** — ตอนขึ้นป้าย ✦ ระบบทักว่า "กิ่งนี้คล้ายกิ่งเดิม จะรวมไหม" กันกิ่งงอกซ้ำ
- 🔴 **I. ส่งเฉพาะหมวดที่เกี่ยวข้อง (ไม่ส่งทุกกิ่ง)** — กัน token บวมตอนกิ่งโต 500-1000
- 🔴 **P. กำหนด "โหมดคิด (thinking/reasoning)" ต่อโมเดล** *(พี่กันถาม 21 ก.ค.)* — **ตอนนี้โค้ดไม่ได้ตั้งเลย** ทุกเจ้าใช้ค่า default (Gemini/GPT-5.x คิดเอง · Claude ไม่เปิด extended thinking) → ควรคุมได้ว่าคิดมาก/น้อย ชั่งคุณภาพ vs ราคา/เวลา

## 🥈 ลำดับ 2 — หลาย AI ช่วยกัน (multi-AI)

> กรอบคิด: ความเห็นต่างระหว่างโมเดล = ตัวเลือกให้เจ้าของคลังเลือก ไม่ใช่ข้อผิดพลาด

- 🔴 **M. Flow หลาย AI แบ่งหน้าที่ (multi-agent)** *(ไอเดียใหม่ 21 ก.ค.)* — ตัวหาความหมาย/ตัวแบ่งวลี คนละเจ้าตามความถนัด
- 🔴 **C. Ensemble** — รันครั้งเดียว หลายเจ้าพร้อมกัน แล้วรวมผล (union + dedupe)
- 🔴 **G. AI ตัวที่ 2 ตรวจซ้ำทั้งช่อ** — เสนอคำสกัดเพิ่ม/หมวดที่ควรเปลี่ยน · 2 เสียงต่าง = ให้ผู้ใช้เลือก
- 🟡 **D. Prompt แยกต่อเจ้า** — มี `PROVIDER_NOTES.gpt` แล้ว · ขยายให้ครบทุกเจ้าถ้าจำเป็น
- *(M + C + G น่ารวมออกแบบเป็นระบบเดียว)*

## 🥉 ลำดับ 3 — ผู้ช่วย + ข้อมูล/สถิติ

- 🔴 **H. แชตในเว็บ** — ถามตอบเรื่องคลัง · สั่งจัดคำผ่านแชต · "เกลาหมวด" (หมวดคำเยอะ→เสนอแตกกิ่ง)
- 🔴 **O. รหัสช่อ (batch code) + เก็บสถิติช่อถาวร** *(ไอเดียใหม่ 21 ก.ค.)* — id + typed/ขาด/สกัด/กิ่ง/เวลา/รุ่น อยู่แม้คำย้ายเข้าคลัง
- 🔴 **เวอร์ชัน ๓ (เทส AI):** เฉลย (E) + รันรุ่นเด่นซ้ำ 3 ครั้ง + เทสตระกูลใหม่ (Claude/DeepSeek/Kimi/GLM · เหลือ Qwen) + ยืนยันว่า H4/prompt พอประมาณ/กิ่งบุคคล ได้ผลจริง
- 🔴 **1. โหมด "จัดคำผ่านแชต" (ไม่ใช้ API)** — ให้ผู้ช่วยอ่านคำเองแล้ว insert เข้าตรวจทาน (โหมดเสริมส่วนตัว)
