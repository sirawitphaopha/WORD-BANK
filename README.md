# คลังคำ (Word Bank)

เว็บส่วนตัวสำหรับนักเขียนนิยาย ไว้เก็บ **คำงาม วลี และประโยคสวยๆ** ที่เจอตอนอ่านนิยาย
ให้ระบบช่วยแก้คำสะกด แยกวลีย่อยจากประโยคยาว และจัดเข้าหมวดให้อัตโนมัติ

ธีมกระดาษวรรณกรรมอุ่นๆ — สีครีม หมึกน้ำตาล ฟอนต์ไทยลายมือและ serif

## ✨ ฟีเจอร์
- **เพิ่มคำ** — วางข้อความที่เก็บมา (คำเดี่ยว วลี หรือประโยคปนกัน) แล้วให้ AI ช่วยจัด
- **ตรวจทาน** — ดูผลก่อนบันทึก แก้ตัวคำ เปลี่ยน/ลากย้ายหมวด เลือกหลายคำจัดการพร้อมกัน (มุมมองการ์ด/ตาราง/คอลัมน์)
- **คลังคำ** — คลังที่บันทึกไว้ เป็น **ต้นไม้หมวดย่อยลึก 3 ชั้น ยุบ-ขยายได้** + **สารบัญกดกระโดด** · ค้นหา กรอง เรียง ปรับแบบการ์ด/ขนาด/ฟอนต์ · ตรวจคำซ้ำ
- ปรับสี ฟอนต์ เลย์เอาต์ได้เอง (จำค่าไว้ให้)

## 🛠 เทคโนโลยี
Next.js 15 + React 19 · Supabase (Postgres) · ตัวจัดคำแบบ heuristic (ออกแบบให้สลับเป็น Claude API ได้ทีหลัง)

## ▶️ เริ่มใช้งาน
```bash
npm install
cp .env.example .env.local     # แล้วเติม SUPABASE_SERVICE_ROLE_KEY
npm run dev                    # เปิด http://localhost:3000
```

## 📝 หมายเหตุ
- ไม่มีระบบล็อกอิน เปิดใช้ได้ทันที · ข้อมูลจริงเก็บบน Supabase (ตารางขึ้นต้น `wb_`)
- การตั้งค่าแสดงผลเก็บในเครื่องผู้ใช้ (localStorage)
- คลังคำตั้งต้น 679 คำ มาจากคลังจริงของนักเขียน
- ดูรายละเอียดโครงสร้าง วิธีต่อ Claude และกฎการทำงานได้ใน [`CLAUDE.md`](./CLAUDE.md)
