# คลังคำ (Word Bank)

เว็บส่วนตัวสำหรับนักเขียนนิยาย ไว้เก็บ **คำงาม วลี และประโยคสวยๆ** ที่เจอตอนอ่านนิยาย
ให้ระบบช่วยแก้คำสะกด แยกวลีย่อยจากประโยคยาว และจัดเข้าหมวดให้อัตโนมัติ

ธีมกระดาษวรรณกรรมอุ่นๆ — สีครีม หมึกน้ำตาล ฟอนต์ไทยลายมือและ serif

## ✨ ฟีเจอร์
- **เพิ่มคำ** — วางข้อความที่เก็บมา (คำเดี่ยว วลี หรือประโยคปนกัน) แล้วให้ AI ช่วยจัด · **เลือกตัว AI + รุ่นได้** (มีโลโก้จริงของแต่ละเจ้า) · **แก้คำสั่ง (prompt) เองได้บนหน้าเว็บ** (สองภาษา) · หน้าโหลดนับวินาที ยกเลิกได้ กันข้อความหายตอนรีเฟรช
- **ตัว AI สลับได้ 8 แบบ** — พื้นฐาน (ฟรีในเครื่อง) + Gemini · Claude · GPT (5.x/4.1) · DeepSeek · Kimi · Qwen · GLM (ใส่กุญแจเฉพาะเจ้าที่ใช้ · มีปุ่ม "🔗 ขอกุญแจ" ลิงก์ไปเว็บผู้ให้บริการ)
- **ส่งออกผลตรวจทาน** — ปุ่ม ⬇ ส่งออกเป็นไฟล์ .txt (คำ/หมวด/ต้นฉบับ/ความหมาย/ที่มา) ไว้เทียบว่า AI เจ้าไหนสกัด+จัดหมวดดีกว่ากัน
- **AI ทำอะไร** — แก้คำสะกด · เก็บประโยคเต็ม + สกัดคำเด่นเพิ่ม (จำได้ว่าสกัดจากประโยคไหน) · แปะชนิด (คำ/วลี/ประโยค) · จัดหมวด + หมวดย่อย · ดักคำซ้ำ
- **ตรวจทาน** — ดูผลก่อนบันทึก แก้ตัวคำ เปลี่ยนชนิด/หมวด เตือนคำซ้ำ (ลบคำซ้ำรวดเดียวได้) · **5 มุมมอง**: การ์ด · ตาราง · คอลัมน์ · จับกลุ่มประโยค · แบบคลังคำ (ต้นไม้หมวด) · **คำตรวจทานเก็บบนคลาวด์ เปิดเครื่องไหนก็เห็นตรงกัน**
- 🌿 **ระบบ "ช่อคำ"** — จัดคำรอบใหม่จะเป็น **ช่อใหม่ต่อท้าย ไม่ทับคำที่ยังตรวจไม่เสร็จ** · เปิดดูทีละช่อผ่านแท็บ (หน้าไม่หน่วงแม้มีหลายร้อยคำ) · บันทึกหรือลบทีละช่อได้ · เตือนคำซ้ำ 2 แบบ: ซ้ำกับคลัง และ **ซ้ำข้ามช่อ** (กดป้ายกระโดดไปดูใบที่ซ้ำได้) · แท็บช่อกับแถบเครื่องมือตรึงไว้ เลื่อนดูคำล่าง ๆ ก็ยังกดได้
- **เกี่ยวกับ** — หน้าแนะนำตัวเว็บ: ผู้พัฒนา · ข้อมูลแอป · สถิติคลังคำแบบสด · AI ที่รองรับพร้อมสถานะกุญแจ · ที่เก็บข้อมูล · เทคโนโลยีที่ใช้
- 🌿 **หมวดย่อยหลายกิ่งต่อคำ** — คำหนึ่งติดได้หลายหมวดย่อยไม่จำกัด (เช่น "ใจสั่นระริก" อยู่ทั้งความรักและความตื่นเต้น) · โผล่ครบทุกกิ่งในต้นไม้ · AI เสนอกิ่งใหม่ได้ ขึ้นป้าย ✦ ให้กดอนุมัติ · กรองดูเฉพาะกิ่งใหม่ได้
- 🛟 **งานไม่หาย** — ผลจาก AI ถูกบันทึกขึ้นคลาวด์ตั้งแต่ฝั่งเซิร์ฟเวอร์ ปิดหรือรีเฟรชหน้าระหว่างรอก็ไม่เสียเงินฟรี
- 🔤 **เลือกฟอนต์ได้ 6 แบบ** รวม **TH Sarabun New** (ฟอนต์ราชการ) และ **Cordia New** (ใช้ได้บนเครื่อง Windows)
- 📌 **ตรึงแถบค้นหา สารบัญ และหัวข้อหมวด** เลื่อนดูคำเป็นร้อยก็ยังรู้ว่าอยู่หมวดไหน + ปุ่มขึ้นบนสุด/ลงล่างสุด
- **คลังคำ** — **ต้นไม้หมวดย่อยลึก 3 ชั้น ยุบ-ขยายได้** + **สารบัญกดกระโดด** · ค้นหา กรอง (หมวด/เรื่อง/**ชนิด**) เรียง ปรับแบบการ์ด/ขนาด/ฟอนต์
- **ประวัติการใช้ AI** — เก็บทุกครั้งที่เรียก AI: เจ้าไหน รุ่นอะไร token ค่าใช้จ่ายประมาณ เวลา สำเร็จ/พลาด + สรุปยอดรวมต่อเจ้า
- 💭 **AI เขียนเหตุผลและความหมายให้** — คำที่เข้าได้หลายหมวดหรือจัดยาก AI จะเขียนเหตุผลสั้น ๆ กำกับว่าทำไมถึงจัดแบบนี้ และใส่ความหมายสั้น ๆ ให้ด้วย เลือกเก็บหรือย้ายเองได้เร็วขึ้น
- 🔬 **ผลทดสอบ AI + 📜 ประวัติคำสั่ง** — 2 หน้ารายงานในเว็บ ดูได้ว่าโมเดลไหนดีกว่ากันด้วยข้อมูลจริง และคำสั่งที่ใช้เปลี่ยนไปอย่างไรบ้าง
- ปรับสี ฟอนต์ เลย์เอาต์ได้เอง (จำค่าไว้ให้)

## 🔬 การทดสอบโมเดล AI

โปรเจกต์นี้มี **ระบบทดสอบโมเดล AI แบบมีมาตรฐาน** เพราะเว็บสลับ AI ได้ 8 เจ้า จึงต้องรู้ว่าเจ้าไหนจัดคำได้ดีจริง

📄 **[`docs/AI-MODEL-TEST.md`](./docs/AI-MODEL-TEST.md)** — เอกสารฉบับเต็ม (มีสรุปสั้นอยู่หัวไฟล์ อ่าน 1 นาทีจบ)
เขียนแบบรายงานวิจัย: วิธีการ (Method) · ผล (Result) · อภิปราย (Discussion) · ข้อจำกัด (Limitations)

- **ชุดคำทดสอบมาตรฐาน = เวอร์ชัน 1** (31 บรรทัด) ใช้ชุดเดียวกันทุกรอบเพื่อให้เทียบผลกันได้จริง
- 🔢 **ชุดคำจะถูกปรับให้แม่นขึ้นในอนาคต** → แก้ชุดคำหรือวิธีวัดเมื่อไหร่ = **ขึ้นเวอร์ชัน 2** และ**เก็บผลเวอร์ชันเดิมไว้เทียบเสมอ** (ผลข้ามเวอร์ชันเทียบกันตรง ๆ ไม่ได้ เพราะเป็นข้อสอบคนละชุด)
- 🖥 **มีหน้ารายงานในเว็บด้วย 2 แท็บ** — 🔬 ผลทดสอบ AI (ตารางเทียบการจัดกลุ่มทีละคำ กางครบทุกคำ) และ 📜 ประวัติคำสั่ง (เทียบทุกฉบับแบบ GitHub เขียว=เพิ่ม แดงขีดฆ่า=ลบ)
- 📐 **วัด 2 มุม**: (๑) สกัดได้แค่ไหน (๒) จัดกลุ่มตรงกันไหม — พบว่าคำที่พิมพ์เข้าจัดตรงกันทุกโมเดลเพียง 29% และคำที่สกัดมา 42% มีโมเดลเดียวที่มองเห็น จึงเป็นเหตุผลที่ควรให้ AI หลายตัวช่วยกันเสนอ แล้วเจ้าของคลังเลือกเอง
- 🛟 **ตาข่ายกันคำหาย** — ระบบนับบรรทัดที่พิมพ์เข้าเทียบผลจาก AI ถ้า AI ตัดทิ้ง ระบบเติมกลับให้พร้อมแจ้งเตือน (ไม่พึ่ง AI เลย)
- ผลรอบที่ 1 (19 ก.ค. 2569): **Gemini 3.1 Pro ดีที่สุด** · Flash ฟรีคุ้มที่สุด · **ตระกูล GPT ทำคำที่ผู้ใช้พิมพ์เข้าหายโดยไม่แจ้งเตือน** และไม่ให้หมวดย่อยหลายกิ่งเลย

## 🛠 เทคโนโลยี
Next.js 15 + React 19 · Supabase (Postgres) · ระบบ AI สลับหลายเจ้า (provider adapter) — กุญแจเก็บฝั่งเซิร์ฟเวอร์ (.env.local) · มีตัวเลือก "พื้นฐาน" (heuristic ในเครื่อง) เป็นตัวสำรองฟรี · **ฟอนต์ไทยฝังในเว็บ** (next/font ไม่พึ่ง CDN) · deploy บน Cloudflare Workers

## ▶️ เริ่มใช้งาน
```bash
npm install
cp .env.example .env.local     # แล้วเติม SUPABASE_SERVICE_ROLE_KEY
npm run dev                    # เปิด http://localhost:3000
```
