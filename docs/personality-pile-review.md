# กองรวมคำที่ย้าย/ควบเข้าหมวดบุคลิกภาพ (รอตรวจก่อนกระจายจริง)

> **นี่คืออะไร:** ไล่เก็บ **ทุกคำที่มาร์กไว้ว่าจะ "ย้าย" 🚚 หรือ "ควบหมวด" 🔗 เข้าหมวดบุคลิกภาพ** (หมวด ๘ นิสัย + หมวดสถานะ) จากทุกหมวดที่เกลาแล้ว มากองไว้ที่เดียว เพื่อให้ตรวจซ้ำก่อนกระจายลงจริง
> **ที่มา:** สแกนตาราง 🚚/🔗 ของทุกไฟล์ redesign (หมวด ๑–๖ · ๑๐) · หมวด ๑/๓/๑๐ ไม่มีคำเข้าบุคลิกภาพเลย
> **สถานะ (อัปเดต):** ✅ agent ตรวจแล้ว + พี่กันเคาะ ๕ ข้อ (ทำตาม agent) + **กระจายคำลงไฟล์ร่างหมวดปลายทางแล้ว** (cat8 · cat11 · cat6 · cat3) · ดูสรุปการกระจายท้ายไฟล์
>
> 🔢 **จำนวน: ๒๔ คำไม่ซ้ำ** — จากหมวด ๕ (๑๔) + หมวด ๖ (๘) + หมวด ๔ (๑) + หมวด ๒ (๒) = ๒๕ รายการ ลบซ้ำ "เหลวไหลไร้สาระ" (อยู่ทั้ง ม.๕/ม.๖) = **๒๔ คำ**
> ⚠️ **"๑๔ คำ" ที่พี่กันจำ = เฉพาะที่ย้ายจากหมวด ๕** (ตาราง 🚚 ของ cat5) · พอรวมที่ควบมาจากหมวด ๖/๔/๒ ด้วย = ๒๔

## 🧭 ๓ แกนที่ใช้แยก (จากงานวิเคราะห์)
- **① Intrinsic — นิสัยแท้** (หมวด ๘) · "เขาเป็นคนแบบไหน" ถอดสถานะออกก็ยังเป็น (ดื้อดึง จองหอง เหี้ยมโหด)
- **② Extrinsic — สถานะ/บทบาท** (หมวดสถานะ) · "เขาเป็นอะไรในสังคม" (ชื่อเสียง ฆาตกร ชาติตระกูล)
- **③ Evaluative — คำตัดสิน/ประเมินค่า** · "คนอื่นรู้สึก/ตัดสินเขายังไง" อยู่ที่สายตาคนมอง (น่าเวทนา น่ารักน่าเอ็นดู) → **แคลร์เก็บเป็นกิ่งติดธงในหมวด ๘ + โยงหมวด ๕**

## 🚩 จุดที่ต้องเคาะ (แคลร์ตั้งข้อสังเกต · รอ agent + พี่กัน)
1. **"ดูแคลน" (#6)** — เป็นกริยา ("ดูถูกเหยียดคนอื่น") ไม่ใช่ป้ายบนตัวคน → น่าจะบ้านหลัก **หมวด ๓ (กริยา)** ไม่ใช่บุคลิกภาพ
2. **กลุ่ม "น่า..." (#8–12) + นังตัวดี (#7)** — เป็น **แกนที่ ๓ (Evaluative · มุมคนมอง)** ไม่ใช่นิสัยแท้ · เก็บกิ่งติดธง ม.๘ + โยง ม.๕ (หรือพี่กันอยากยกเป็นหมวด/แกนแยก)
3. **ควบอ่อน ๓ คำ (#23–25)** — ถ้อยคำถ่อยสถุล · มืออันหยาบกร้าน · อ่อนปวกเปียก = คำที่บ้านหลักอยู่หมวดอื่นแล้ว "สื่อถึงนิสัย" ทางอ้อม → รอ agent ประเมินว่าควบเข้าบุคลิกจริงไหม หรือควบหลวมเกินไป
4. **ก้ำกึ่ง Intrinsic/Evaluative (#5, #13, #14)** — โหดเหี้ยมเลือดเย็นฯ · แปลกพิกล · เหลวไหลไร้สาระ = อยู่ระหว่างนิสัยกับการตัดสิน
5. **คำที่แตกเป็นหลายแกน (#19, #22)** — ฆาตกรกระหายเลือด · เชื้อไม่ทิ้งแถว = ติดทั้ง ม.๘ และหมวดสถานะ

## ✅ งานที่ให้ agent ตรวจ
- **ครบไหม** — มีคำที่ย้าย/ควบเข้าบุคลิกภาพในหมวดอื่นที่ตกหล่นไหม (สแกนซ้ำทุก redesign)
- **แกนถูกไหม** — แต่ละคำเข้าแกน ①/②/③ ถูกไหม · มีคำไหนไม่ควรเข้าบุคลิกเลย (เช่น กริยา)
- **กิ่งถูกไหม** — กิ่งที่เสนอในหมวด ๘/สถานะ เหมาะไหม
