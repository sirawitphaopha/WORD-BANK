### 2) 🌟 เมนูเลือก AI มือถือจัด 2 แถว (layout นี้ **พี่กันกำหนดเอง** — แก้ 2 รอบกว่าจะตรงใจ)
**ปัญหาเดิม:** บนมือถือ ช่องเลือกเจ้า + ช่องเลือกรุ่น + ปุ่มขอกุญแจ อยู่แถวเดียว flex-wrap → พอเลือก **GPT** ชื่อรุ่นยาว (`GPT-4.1 · ★★☆☆☆ 2.5 สกัดดีแต่เคยทำคำหาย 3`) **ดันจนล้นจอ เลื่อนซ้าย-ขวาได้**
**สิ่งที่พี่กันสั่ง (คำพูดจริง):** "api key อยู่แถวเดียวกันกับที่เลือกว่าใช้ AI ตัวไหน · ส่วนโมเดลอะไร ก็แถบยาวเหมือนเดิม"
**layout สุดท้าย (เฉพาะมือถือ):**
- **แถว 1** = ป้าย "ผู้ช่วย AI" + โลโก้ (`<BrandIcon>`) + **ช่องเลือกเจ้า** (`{flex:1, minWidth:0}` หดพอดีจอ) + **ปุ่ม api key** (`{flex:'none'}`) ← api key อยู่แถวเดียวกับช่องเลือก**เจ้า**
- **ตัวเว้นบรรทัด** `<div style={{flexBasis:'100%', height:0}} />` บังคับขึ้นแถวใหม่ (เทคนิค flex-wrap)
- **แถว 2** = **ช่องเลือกรุ่น** (`{flex:1}`) → อยู่คนเดียวในแถว = **ยาวเต็มความกว้าง โชว์ชื่อรุ่นเต็ม ไม่ต้องหด ไม่ตัด … ไม่ล้นจอ**
- 🔑 **จุดสำคัญของโค้ด:** ปุ่ม api key **เรนเดอร์ 2 ชิ้นแยกกัน** ด้วย guard `S.isMobile`:
  - มือถือ: `{S.isMobile && cur.keyUrl ? <a ... flex:'none'>🔗 api key</a> : null}` อยู่**หลังช่องเลือกเจ้า** (แถว 1)
  - จอคอม: `{!S.isMobile && cur.keyUrl ? <a ...>🔗 api key</a> : null}` อยู่**ท้ายสุดตำแหน่งเดิม**
  - เหตุผล: จอคอมต้องการลำดับเดิม (เจ้า → รุ่น → api key) · ถ้าย้าย element เดียวไปแถวบนจะทำให้จอคอมลำดับเพี้ยน → เลยแยกเป็น 2 ชิ้น เรนเดอร์ทีละจอ → **จอคอม DOM เหมือนเดิมเป๊ะ**
- **เลิกใช้** `minWidth:0` ที่ช่องเลือกรุ่น (เดิมใส่เพื่อหด+ตัด …) เพราะตอนนี้รุ่นอยู่แถวเดียวยาวเต็มแล้ว ไม่ต้องหด

### 3) ปุ่มขอ API key: "🔗 ขอกุญแจ" → "🔗 api key"
- พี่กันขอย่อคำ → เปลี่ยนทั้ง 2 จอ · เพิ่ม tooltip `title="เปิดเว็บขอ API key ของ <ชื่อเจ้า>"`

### 4) ตัวนับกล่องวางคำ: "ยังว่างอยู่/N บรรทัด" → "N คำ/วลี" + ย้ายที่
- เดิมโชว์ `ยังว่างอยู่` หรือ `N บรรทัด · M ตัวอักษร` ในแถวปุ่ม — พี่กันถามว่า "ยังว่างอยู่ มีไว้ทำไม" + "ใส่เลข 0 · ใช้ คำ/วลี แทนดีกว่ามั้ย"
- แก้: ตัวแปรเดิม `lineHint` → **`countHint = "<N> คำ/วลี"`** (นับเฉพาะบรรทัดที่มีข้อความ · ว่าง = "0 คำ/วลี")
- **ย้ายไปอยู่ข้างป้าย "ข้อความที่เก็บมา"** (แถว flex: label ซ้าย + ตัวเว้น flex:1 + countHint ขวา) ทั้ง 2 จอ

### 5) placeholder กล่องวางคำ (มือถือ)

### 6) ปุ่ม "วาง" + "ให้ AI ช่วยจัด" (มือถือ)
- พี่กันขอ "ย้ายมากลาง + ทำใหญ่ ๆ เหมือนปุ่มตกลง"
- มือถือ: `flexDirection:'column'` เต็มความกว้าง (`width:100%`) จัดกลาง · ปุ่ม "ให้ AI ช่วยจัด" = **ปุ่มหลักใหญ่สุด** (padding 15–19px, font 17–20px, weight 700, เงาเข้ม)
