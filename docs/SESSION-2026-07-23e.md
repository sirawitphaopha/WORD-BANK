## บทเรียนสำคัญ (ห้ามซ้ำ)
1. 🔴 **คำจริง = ชิปในกิ่ง · ความเห็น/ตัวอย่าง/ใครเสนอ = อยู่บรรทัด 🗨 หมายเหตุเท่านั้น ห้ามเป็นชิป**
   - เคยเผลอใส่ `ชก` `ต่อย` `เตะ` เป็น "คำเพิ่ม" (ชิป) ในกิ่งการต่อสู้หมวด 3 → พี่กันท้วง "มันคือความเห็นนี่นา / เอาออกนะ"

2. 🔴 **`[ใคร]` vs `[ชื่อคน]` ในคำต้นฉบับ** — คำ "ด่าทอ...อย่างสาดเสียเทเสีย" เป็นคำต้นฉบับที่มี "..." (จุดไข่ปลา = ช่องเว้นที่พี่กันตั้งใจ ตอน import ถูกตัดออก)

   - กฎ: "..." ในต้นฉบับแทนด้วยวงเล็บเหลี่ยม `[ชื่อคน]/[สิ่งของ]/[สถานที่]/[...]` ตาม `lib/prompt.js` (แคลร์แบ่งคำ = ต้องยึด prompt เดียวกับ AI)
