ฆาตกรกระหายเลือด๗ ป้ายศีลธรรม (ฉายา)หมวด ๖ + ๘“ฆาตกร” = บทบาท (สถานะ) + “กระหายเลือด” = นิสัย (๘)
ยังไม่มีคำบ้านหลัก — หมวดนี้มีแต่คำโยงเข้า ๓ คำ · คำตัวอย่างในกิ่ง (ดอกเตอร์/นางคณิกา/สุจริตชน) เป็นตัวอย่างตั้งเผื่อ ยังไม่ใช่คำในคลัง

🗣 อภิปราย

✅ รับจากเจน: แนวคิด Character Design แยก “หัวโขน/ป้ายแขวนคอ” (Extrinsic) ออกจาก “เนื้อแท้/ซอฟต์แวร์” (Intrinsic) — คมและใช้ได้จริงในนิยาย

🔧 แคลร์เสนอเพิ่ม (ต่างจากเจน): กิ่ง ๘ “บทบาทนอกกรอบและการสังกัด” — ทาส/เชลย/คนนอกกฎหมาย/สมาชิกสมาคม ไม่มีที่ลง · “คนไร้ราก” (คนพเนจร) เป็น archetype สำคัญ → ตั้งกิ่งรับ

🚩 เส้นที่ต้องระวัง (Extrinsic vs Evaluative): เจนมีแนวโน้มดึงคำ “ตัดสิน” (ดูแคลน/น่าเวทนา/นังตัวดี) เข้ากิ่ง ๗ · แคลร์แย้ง: คำพวกนี้เป็น “อารมณ์ของผู้พูด” (จริงเฉพาะในสายตาคนมอง) ไม่ใช่ “สถานะที่สังคมนิยามร่วมกัน” → คนละธรรมชาติ อย่าดูดเข้ามาหมด · โยงมากิ่ง ๗ เฉพาะคำที่กลายเป็นป้ายติดตัวจริง (ฆาตกรกระหายเลือด)

⚡ multi-branch เยอะเป็นปกติ: ดอกเตอร์ (อาชีพ + คำนำหน้า) · เจ้าขุนมูลนาย (ยศ + ชนชั้น) · ผู้ดีแปดสาแหรก (ชนชั้น + ชาติตระกูล)
