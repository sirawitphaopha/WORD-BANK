## 🔑🔑 กฎใหม่ 3 ข้อ (พี่กันตั้งรอบนี้ · ใช้ทุกหมวดต่อจากนี้ + จะลง prompt AI)
บันทึกใน CLAUDE.md แล้ว · ทำซ้ำที่นี่เพื่อความชัด:

1. **📛 กฎชื่อกิ่ง** — ใช้ **คำธรรมดาที่สื่อตรงที่สุด ไม่ใช้คำสวย/วรรณกรรม** + ใส่คำบริบท (กิ่งใต้ดวงตามีคำว่า "ตา" · กิ่งใต้ผมมีคำว่า "ผม")
   - เหตุผล: **ชื่อกิ่ง = ป้ายให้ AI จัดคำ ต้องชัด**
   - ตัวอย่างที่แก้: งดงาม→หน้าตาสวยงาม · ประกายและชีวิตชีวา→แววตาและประกายตา · ซูบซีดโรยรา→ใบหน้าซีดเซียวและทรุดโทรม

2. **📖 กฎนิยาม** — นิยาม (— _..._) = **ความหมายจริงของกิ่ง (กิ่งนี้ครอบคลุมอะไร) เขียนตรง ๆ สะอาด ไม่ใช่ list คำตัวอย่าง** เพราะขึ้นเว็บจริง
   - ตัวอย่างคำ/เหตุผล/ใครเสนอ = แยกไปบรรทัด **🗨 หมายเหตุ** (เก็บไว้เสมอ ไม่ลบ · บอกว่ามีกิ่งนี้เพราะอะไร)
   - พี่กันเน้น: "บางอันมันไม่ใช่ความหมาย มันคือตัวอย่าง นี่ก็กฎด้วยนะ"

3. **🏷 กฎ tag แนวนิยาย** ([แฟนตาซี][ไซไฟ][อีโรติก][ทริลเลอร์][ทั่วไป]) — ใส่ในรายงาน/md/docs ได้ · **ตอนขึ้น Supabase/เว็บ/ส่ง AI = เอาออก**
   - เหตุผล: เจนเตือน bias — AI เห็นกิ่ง "อมนุษย์" ติด [ไซไฟ] แล้วลังเลจัด "ปีศาจ" จากนิยายสยองลงกิ่งนี้ · พี่กันเห็นด้วย

**+ หลักคิดที่พี่กันย้ำ (ไม่ใช่กฎใหม่ แต่ต้องจำ):**
- **แก้คำสะกดผิดได้** (ไม่ใช่ลบวลี) — ต้อง detect ได้เอง
- **"คลังต้องรับได้กับคำทุกรูปแบบในทุกแนวนิยาย"** → ตั้งกิ่งเผื่อทุกแนวแม้ยังไม่มีคำ
- **"โยงได้หมด ไม่จำเป็นต้องความหมายซ้ำหมวดเดียวกัน"** → คำติดกิ่งข้ามหมวดได้ 🔗
