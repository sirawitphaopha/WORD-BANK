🔗 คำโยงข้ามหมวด (คงบ้านหลักเดิม + โผล่หมวด ๘)

ประเมินแล้วว่าอยู่หลายหมวดจริง — คงบ้านหลักเดิมไว้ + โผล่ในหมวด ๘ ด้วย (ไม่ยกทิ้ง · พี่กัน “โยงได้หมด”)
คำกิ่งในหมวด ๘บ้านหลักเหตุผล
ดันทุรัง๕ ดื้อรั้นหัวแข็งหมวด ๖คำขยาย (๖) + รากคือนิสัยดื้อ (๘)
ประมาทเลินเล่อ๗ ประมาทเลินเล่อหมวด ๖คำขยาย (๖) + นิสัยสะเพร่า (๘)
เหลวไหลไร้สาระ๗ เหลวไหลไม่เอาไหนหมวด ๕/๖ตัดสินลักษณะ + นิสัยเหลวไหล (๘)
ฆาตกรกระหายเลือด๑ โหดร้าย + ๑๑ คำด่าหมวด ๖ฉายา (๖) + นิสัยโหด (๘) + บทบาท (สถานะ)
ตาถั่ว๑๐ มองคนไม่เป็นหมวด ๖สำนวน (๖) + คุณสมบัติถาวร (๘)
นิสัยใจคอ๑๐ นิสัยใจคอโดยรวมหมวด ๖คำ meta (๖) + ชื่อของ “นิสัย” เอง (๘)
เชื้อไม่ทิ้งแถว๑๐ นิสัยสืบสายเลือดหมวด ๖สำนวน + นิสัยสืบทอด (๘) + ชาติตระกูล (สถานะ)
เสื่อมเสียควบ “หมวดสถานะ” กิ่งชื่อเสียงหมวด ๖ชื่อเสียงมัวหมอง = สถานะสังคม (ไม่ใช่หมวด ๘)
ถ้อยคำที่ถ่อยสถุลระดับภาษาบ่งบุคลิกหมวด ๔ลักษณะคำพูด (๔) + ระดับภาษาสะท้อนนิสัย (๘)
มืออันหยาบกร้าน · อ่อนปวกเปียกกายบ่งนิสัยหมวด ๒สภาพกาย (๒) + สื่อชีวิต/นิสัย (๘)

🗣 ประเมิน + อภิปราย

✅ รับ: แกน Intrinsic/Extrinsic ของเจนคมและใช้ได้จริง · จัดกิ่งตาม “ขั้วนิสัย” (๒ ขั้วในร่มเดียว) ทำให้ AI จัดคำบวก-ลบในกิ่งเดียว

🚩 แกนที่ ๓ (คำตัดสิน/ประเมินค่า): ๓ มุมชี้ตรงกันว่า “น่าเวทนา/น่ารักน่าเอ็นดู/นังตัวดี” เป็น มุมของคนมอง (POV) ไม่ใช่นิสัยแท้ → ถ้าปนกับ “ดื้อดึง” หมวดจะเลอะ · แคลร์เคารพที่พี่กันเคยเคาะให้อยู่หมวด ๘ → เก็บเป็นกิ่งติดธง ๑๑ แยกชัด + โยงหมวด ๕ ทั้งกลุ่ม (น่าเวทนา🔗เวทนา · น่าคลื่นเหียน🔗รังเกียจ · น่ารักน่าเอ็นดู🔗รัก) · ถ้าพี่กันอยากยกเป็นแกน/หมวดที่ ๓ หรือย้ายบ้านหลักไปหมวด ๕ บอกได้

🚩 “ดูแคลน” หลุดกลุ่ม (แคลร์จับเพิ่ม): จริง ๆ เป็น กริยา (“ดูถูกเหยียดคนอื่น”) ไม่ใช่ป้ายบนตัวคน → บ้านหลักน่าจะ หมวด ๓ (กริยา) + โยงหมวด ๘ (นิสัยชอบดูถูก) · พักไว้กิ่ง ๑๑ ก่อน รอพี่กันเคาะ

🔧 กิ่งเผื่อจากมุมคนเขียนนิยาย (แคลร์เสนอ): กิ่ง ๑๒ ภูมิหลัง/ปมในใจ + กิ่ง ๑๓ แรงขับ/ความมุ่งหมาย = แกนสร้าง character arc ที่คลังยังไม่มีบ้าน · ตั้งเป็นกิ่งเผื่อก่อน (ยังไม่ยกเป็นหมวด)

⚡ multi-branch ที่ต้องคง: ฆาตกรกระหายเลือด (โหดร้าย + ฉายา + บทบาท) · เชื้อไม่ทิ้งแถว (นิสัยสืบทอด + ชาติตระกูล) · สงบเสงี่ยม (ถ่อมตน + ใจเย็น)
