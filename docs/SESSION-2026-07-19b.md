## 📌 ตรึง (sticky) ในหน้าคลังคำ

1. **แถบค้นหา + ตัวกรอง + ชิปหมวด** → ตรึงเป็นก้อนเดียว `top:69px` (69 = ความสูงแบนเนอร์)
2. **สารบัญ (aside)** เดิมตรึงที่ 74px ซึ่ง**โดนแถบด้านบนทับ** (แถบกินถึง 172px)
   → เปลี่ยนเป็น `tocTop = 69 + libStickH + 10` โดย `libStickH` วัดจากของจริงด้วย ResizeObserver
   (`_libStickRef` + state `libStickH`) ถ้าแถบสูงขึ้นเพราะเพิ่มตัวกรอง สารบัญจะขยับหลบเอง
3. **หัวข้อหมวดตรึงแบบเปลี่ยนหัวตามหมวด** — ใส่ `position:sticky` บนหัวข้อหมวด
   sticky ทำงานในขอบเขตกล่องหมวดของตัวเอง → พอเลื่อนพ้นหมวด หัวข้อใหม่ดันหัวเก่าออกแล้วค้างแทน
   (พฤติกรรมเดียวกับรายชื่อผู้ติดต่อในมือถือ) · เปลี่ยนพื้นเป็นทึบ + เพิ่มเงา กันคำด้านหลังทะลุ
